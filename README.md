
# oopMiniProject

This is a Laravel-based web application developed as an OOP Mini Project.

## Technologies Used
- Laravel
- PHP
- MySQL
- HTML, CSS, JavaScript
- Git & GitHub

## Setup Instructions
1. Clone the repository
2. Run `composer install`
3. Run `npm install`
4. Copy `.env.example` to `.env`
5. Configure database settings in `.env`
6. Run `php artisan key:generate`
7. Run `php artisan migrate`
8. Run `npm run dev`
9. Run `php artisan serve`
